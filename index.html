Модуль 1
PRE-PARTY
Уроки:
1
Знайомство та організаційні моменти навчання
2
Види пензлів: для чого кожен з них потрібен та як правильно дезінфікувати їх
3
Базовий кейс для макіяжу: косметика + інструменти (пензлі, спонжі тощо). + гайд косметичка макіяж для себе
4
Практика: Макіяж для себе на кожен день
Результат:
1. Збереш базову косметичку для створення різних макіяжів — без зайвого, тільки musthave.
2. Зрозумієш, які пензлі потрібні саме тобі, як ними користуватись та як їх очищати.
3. Навчишся робити свій ідеальний макіяж на кожен день.

МОДУЛЬ
КОДЕКС ВІЗАЖИСТА
Уроки:
1
Як стати впізнаваним візажистом

Кодекс Візажиста:: організація робочого місця та портрет етичного візажист
3

Аналіз конкурентів та пошук ідей для контенту
БОНУСНИЙ УРОК ДЛЯ ВСІХ ВІД ПСИХОЛОГА:
Як побороти страхи та почати діяти
Результат:
1. Отримаєш основи роботи професійного візажиста, до якого повертаються та рекомендують: від організації робочого місця до роботи з клієнтами.
2. Усвідомиш, яким має бути етичний візажист.
3. Зможеш з легкістю знаходити моделей для відпрацювання макіяжів і реалізації всіх своїх ідей.
Модуль 2
СТВОРЕННЯ ІДЕАЛЬНОГО ТОНУ
Уроки:
1
Підготовка шкіри до макіяжу + демакіяж. Як визначити тип шкіри + гайд засоби для різних типів шкіри
2
Як підібрати тон для будь-якого типу шкіри. Види тональних основ, пудр, фіксаторів, консилерів (які засоби для якого типу підходять)
3
Форма та типажі обличчя. Все про контуринг та візуальну зміну обличчя. Як зменшити ніс, зробити виразні скули та інші секрети
4
Макіяж ідеальний нюд
БОНУСНИЙ УРОК З КОСМЕТОЛОГОМ-ДЕРМАТОЛОГОМ:
Як визначити свій тип шкіри. Домашній догляд за шкірою на будь-який бюджет
Результат:
1. Навчишся розбиратися в типах шкіри та підбирати ідеальні відтінки для будь-якого типу.
2. Зможеш створювати рівну, природну шкіру без маски — так, щоб і на фото, і в житті це виглядало дорого.
3. Опануєш техніки для зміни форми обличчя: зменшити ніс, зробити чіткі скули, "підтягнути" брову і освіжити погляд — за 10 хвилин.
